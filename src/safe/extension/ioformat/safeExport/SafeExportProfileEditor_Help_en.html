Editeur de profils d'exportation

Objectif

Permettre l'édition de profils d'exportation

Description

La boîte de dialogue est composée de 3 zones, qui sont (de haut en bas) :

  1. Nom du fichier-profil : ce nom s'affichera dans la boîte de dialogue "Catalogues des profils d'exportation" ; Veillez à utiliser un nom de fichier valide, c'est à dire sans caractère non autorisés tels que *, /, \ ...
  2. Etendue : cette zone permet de définir les étapes concernées par l'exportation ; Attention : lors de l'exportation proprement dite, l'étendue effective dépent de l'étape à partir de laquelle la procédure d'exportation a été appelée.
  3. Variables : cette zone offre autant d'onglets que de sujets disponibles pour l'exportation.

    Chaque onglet contient au minimum :

    • une case à cocher "A exporter" permettant (entre autres), la sélection de variables dans la liste
    • une liste de variables en rapport avec le sujet
    • des boutons de commande permettant de recharger la liste des variables (telle qu'elle est enregistrée dans le fichier profil), de sélectionner toutes les variables, d'en sélectionner aucune ou d'inverser la sélection.

    De plus des informations spécifiques à certains sujets sont disponibles, soit :

    • Onglet "Arbres" : la case à cocher "tout" si elle est cochée, permet d'exporter les variables pour tous les arbres présents dans le scénario ; si elle n'est pas cochée, la zone de texte située à sa droite permet de définir sous forme de liste (entiers séparés par des ;) les identifiants des arbres pour lesquels on veut exporter les variables.
    • Onglet "Cellules" : la case à cocher "tout" qui fonctionne de la même manière que celle décrite pour l'onglet "Arbres" ; pour la sélection de cellules (si "tout" est décoché), un bouton de commande situé à droite de la zone de texte permet de sélectionner graphiquement les identifiants des cellules à exporter.
    • Onglet "Voxels" : la case à cocher "tout" qui fonctionne de la même manière que celle décrite pour l'onglet "Cellules". Dans la zone de texte "profondeurs", il est possible de définir une liste de valeurs discrètes ou sous la forme d'intervalles comme le montre l'exemple suivant :

    Exemple de valeur du champ "Profondeur" : -3.2 ; 5.0 ; 7.1-7.5 ; 8.25-
    -3.2 sélectionne les voxels dont la profondeur est inférieure ou égale à 3.2 mètres
    5.0 sélectionne les voxels dont la profondeur est de 5.0 mètres
    7.1-7.5 sélectionne les voxels dont la profondeur est comprise entre 7.1 et 7.5 mètres
    8.25- sélectionne les voxels dont la profondeur est supérieure ou égale à 8.25 mètres

Concernant la sélection de variables dans la liste, veillez à maintenir la touche "Control" enfoncée lors de la sélection de variables, cette touche permettant d'étendre ou de réduire la sélection existante.
