Catalogues des profils d'exportation
Aide Capsis
(Export Safe

Objectif

Permettre l'exécution et/ou l'édition (création / modification / suppression) de profils d'exportation

Description

La boîte de dialogue est composée des éléments suivants :

  • à gauche, une liste qui contient les profils d'exportation disponibles
  • à droite, une liste de boutons de commande permettant d'exécuter les opérations suivantes :
    • Exécuter... permet d'exécuter l'exportation, (après confirmation), en utilisant le fichier-profil sélectionné dans la liste ;
    • Nouveau... permet de créer un nouveau fichier-profil ;
    • Modifier... permet de modifier le fichier-profil sélectionné dans la liste. Pour créer un nouveau fichier-profil à partir d'un fichier existant, utilisez ce bouton, puis dans la boîte de dialogue "Editeur de profils d'exportation", changez le nom du fichier-profil ;
    • Supprimer... permet de supprimer le fichier-profil sélectionné dans la liste (après confirmation) ;
    • Annuler annule la procédure en cours
    • Aide affiche ce fichier...
