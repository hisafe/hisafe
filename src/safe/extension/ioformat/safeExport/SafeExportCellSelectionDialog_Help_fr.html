Sélection de cellules de terrrain

 Aide Capsis
(Export Safe)

Principe

Sélectionnez les cellules d'intérêt directement avec la souris. Les cellules sélectionnées apparaissent en rouge.

Sélection / désélection

  • Pour (dé)sélectionner une cellule, double-cliquez dessus avec le bouton gauche.

  • Pour (dé)sélectionner un groupe de cellules : définissez un rectangle de sélection englobant avec le bouton droit.

Information sur une ou plusieurs cellules

  • Sélectionnez la ou les cellule(s) en maintenant la touche Control enfoncée.

Zoom

  • Zoom avant : définissez un rectangle de zoom englobant la zone d'intérêt avec le bouton gauche.

  • Zoom arrière : cliquez sur l'image avec le bouton droit.

Déplacement sur l'image

  • Tracez un vecteur de translation sur l'image en maintenant la touche Control enfoncée.
